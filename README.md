# AbdurRaheem Store

A modern e-commerce website built with Next.js, featuring leather products and sports equipment.

## Features

- Product catalog with hierarchical categories
- Responsive design with Tailwind CSS
- TypeScript for type safety
- App Router for modern Next.js routing
- ESLint for code quality

## Categories

### Leather Products
- Leather Bags (Handbags, Shoulder Bags, Tote Bags, Clutches, Travel Bags, Duffel Bags, Laptop Bags, Messenger Bags, Backpacks, Wallets, Card Holders, Passport Holders)
- Leather Jackets (Men's: Biker, Bomber, Classic; Women's: Fitted, Long Coats, Cropped)
- Leather Accessories (Belts, Gloves, Keychains, Watch Straps, Phone Cases, Sunglass Cases)
- Leather Footwear (Men's: Formal Shoes, Casual Shoes, Boots; Women's: Heels, Flats, Boots, Sandals)
- Travel & Utility Leather (Luggage Bags, Weekender Bags, Toiletry Bags, Document Holders)

### Sports Products
- Sportswear (Men's: T-Shirts, Tracksuits, Shorts, Gym Wear; Women's: Leggings, Sports Bras, Training Sets; Kids Sportswear)
- Fitness Equipment (Dumbbells, Resistance Bands, Yoga Mats, Kettlebells, Skipping Ropes, Ab Rollers, Foam Rollers)
- Outdoor Sports (Football, Cricket Equipment, Bats, Balls, Pads, Basketball, Badminton, Volleyball, Table Tennis)
- Gym Accessories (Gym Gloves, Lifting Belts, Wrist Wraps, Knee Support, Elbow Support)
- Sports Footwear (Running Shoes, Training Shoes, Football Shoes, Indoor Court Shoes)
- Bags & Carriers (Gym Bags, Sports Backpacks, Kit Bags)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build the project:

```bash
npm run build
```

## Learn More

This project uses Next.js 15 with the App Router. For more information, see the [Next.js documentation](https://nextjs.org/docs).
